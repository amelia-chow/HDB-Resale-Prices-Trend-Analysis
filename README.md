# 🏠 HDB Resale Price Analysis
An analysis of HDB Resale Prices  

## Project Overview 
This mini project delivers an interactive data visualisation dashboard that allows to analyse and uncover trends, spatial patterns and pricing drivers within the Singapore Housing and Development Board (HDB) resale market. Leveraging over a decade of historical transaction data from [Data.gov.sg](https://data.gov.sg/datasets?topics=housing&resultId=d_8b84c4ee58e3cfc0ece0d773c8ca6abc), the project transforms raw property records into actionable market insights. 

## Project Workflow 

### 1. 🧹Data Cleaning (Python)
- Checked for missing values
- Applied floor division mathematically to round individual remaining lease durations down to nearest decade base

### 2. 📈 Data Visualiation (Tableau)
- Historical HDB prices by year and flat type
- Core drivers of HDB prices 
- Forecast of HDB average prices

[For Access to Dashboard](https://public.tableau.com/shared/ZJ7QW5NPP?:display_count=n&:origin=viz_share_link)

## 📌Key Insights
- Average Resale Prices climb consistently over time across all flat types
- 4-Room flats consistently dominating Resale Market Volume, reprenting the largest single market share at 42.41%
- Floor Area, Remaining Lease, Storey Range and Town affects the HDB Resale Price
- Historical data shows a consistent, upward trajectory for average HDB resale prices over the last decade
